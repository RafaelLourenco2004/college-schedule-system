Bem-vindo à documentação da nossa API do Sistema de Horários Universitários! Esta API foi desenvolvida para facilitar a gestão acadêmica, permitindo que alunos e administradores tenham acesso a funcionalidades essenciais para a organização e administração das atividades educacionais.

Através desta API, os alunos podem:

Gerenciar e configurar suas notas e grades de horários;
Consultar informações sobre suas disciplinas e horários de aula;
Manter-se atualizados com as alterações no cronograma acadêmico.
Já os administradores têm a capacidade de:

Criar e gerenciar lições, matérias e disciplinas;
Organizar e atualizar o cronograma de aulas e eventos acadêmicos;
Automatizar processos administrativos para otimizar a gestão do ambiente universitário.
Esta documentação tem como objetivo fornecer todas as informações necessárias para integrar e utilizar as funcionalidades do sistema, com exemplos, endpoints, parâmetros e respostas da API. Se você é um desenvolvedor buscando implementar soluções que melhorem a experiência acadêmica, ou um administrador encarregado de gerenciar a rotina escolar, esta API foi feita para atender às suas necessidades de forma eficiente e escalável.
